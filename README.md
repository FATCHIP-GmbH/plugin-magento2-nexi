Licence GPLv3

## About Magento 2 Nexi Payment Connector
For details about the plugin please visit [our website](https://www.fatchip.de/Plugins/Magento/Magento-2-Nexi-Payment-Connector.html).

In order to use Nexi Payments via this plugin, you'll need a contract with Nexi. Please contact sales via the form on ADD LINK HERE. In case you are already a customer of Nexi, we kindly ask you to get in touch with your contact person. Following that, you'll receive your access data with which you'll be able to use the Magento 2 plugin. You will then be able to to use Nexi payments in your shop.


## Installation and usage
Visit our Wiki pages to read the [plugin documentation](https://wiki.fatchip.de/public/magento2-fatchipnexipayment).
